# puzzle-box

![logo.png](docs/logo.png)


```bash

# set-up git hooks, .gitignore and README
$ puzzle-box setup-repository

# create files for advent-of-code, 2015, day 1 using c
$ puzzle-box create aoc 2015 1 c

# run in watch mode
$ puzzle-box run -w

# format code using linters
$ pb format

# static code analysis
$ pb check

```
